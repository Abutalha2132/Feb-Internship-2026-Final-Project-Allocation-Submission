##  Project Description

This project is a fully functional backend application built using **FastAPI**.  
It simulates a real-world medical appointment system where patients can book appointments with doctors, manage queues, and perform various operations like search, filtering, sorting, and pagination.

The project demonstrates strong backend development skills, including API design, validation, workflows, and data handling.
